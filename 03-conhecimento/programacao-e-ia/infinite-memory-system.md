# The Infinite Memory System: Integrating NotebookLM, Claude, and Obsidian


> Fonte: notebook do NotebookLM (`The Infinite Memory System: Integrating NotebookLM, Claude, and Obsidian`).  
> Extraído em: 2026-06-18

## TL;DR
Sistema que integra **NotebookLM**, **Claude** e **Obsidian** para resolver o problema da "amnésia" da IA. Usa NotebookLM como base de conhecimento, Claude como processador e Obsidian como memória de longo prazo.

## Problema central
Modelos como o Claude perdem todo o contexto ao abrir uma nova sessão. O objetivo é criar uma **"Biblioteca de Alexandria da IA"**, onde o conhecimento se acumula e o assistente "sabe tudo sobre você" sem reexplicações.

## Os 3 pilares
- **NotebookLM (base):** entrada de vídeos, podcasts e PDFs. Limitação: conhecimento fica preso no aplicativo.
- **Claude (cérebro):** executa tarefas complexas via skills customizadas, acessando o NotebookLM.
- **Obsidian (memória):** armazena conhecimento em Markdown local, facilitando leitura por IA.

## Configuração
1. Instalar skill no Claude via comando, upload ZIP ou `Customize > Habilidades`.
2. Autenticar conta Google do NotebookLM.
3. Criar vault no Obsidian e dar acesso ao Claude.

## Fluxo prático
1. Usar extensão **"YouTube to NotebookLM"** para criar notebook do vídeo.
2. No Claude, usar a skill para listar notebooks (`/notebooklm list`).
3. Transferir conhecimento do NotebookLM para o Obsidian com comando de ponte.
4. A skill formata o conteúdo em Markdown com gráficos e estrutura visual.

## Regras
- Evitar acumular conhecimento só no NotebookLM por causa do limite de créditos.
- Usar Markdown no Obsidian para escaneabilidade por IA.
- O conhecimento é fiel à transcrição original.
