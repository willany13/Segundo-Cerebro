# Skills

Mapa de skills instaladas e como elas se conectam com o vault.

## Instaladas

- [[02-projetos/skill-hermes.md]] — skill do agente Hermes para o projeto Segundo Cérebro
- [[02-projetos/skill-nota-corretagem.md]] — pipeline de notas de corretagem: extração, enriquecimento e auditoria
- notebooklm — NotebookLM CLI
- `notebook-to-md` — converter notebook do NotebookLM em nota estruturada no Obsidian
- `yt-to-notebook` — extrair conhecimento do NotebookLM e transformar em nota visual/digerível com checagem de duplicidade (Fase 0 obrigatória)

## Ligações

- **notebooklm + notebook-to-md**
  - aplicação: `03-conhecimento/programacao-e-ia`
  - uso típico: transformar saída do NotebookLM em nota com TL;DR, diagramas, callouts e links

- **Skill Nota-Corretagem**
  - aplicação: `03-conhecimento/mercado-financeiro`
  - uso típico: extrair PDF de nota → CSV enriquecido → relatório final com DARF estimada

- **Skill-Hermes**
  - aplicação: `02-projetos/skill-hermes.md`
  - uso típico: agente Hermes como skill no opencode para gestão do vault e assistência em projetos

- **second-brain**
  - aplicação em todo o vault: roteamento, `index.md`, captura, limpeza e links cruzados
