# Templates de nota

| Template | Uso |
|----------|-----|
| `captura-rapida.md` | Registrar informação nova do dia |
| `nota-conhecimento.md` | Documentar conceito ou aprendizado |
| `projeto.md` | Estruturar um novo projeto |

## Como usar

1. Copie o template desejado para o pilar correto
2. Preencha os campos entre `{{}}`
3. Remova os placeholders não utilizados
4. Atualize `index.md` e crie links cruzados
